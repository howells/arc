# Plan Mode

Rules for when Claude is in plan mode.

## Concision

- MUST: Make plans extremely concise
- SHOULD: Sacrifice grammar for brevity
- NEVER: Write verbose explanations in plans

## Structure

- MUST: End each plan with "Unresolved Questions" section
- SHOULD: List any ambiguities, missing info, or decisions needed
- SHOULD: If no questions, state "None"
